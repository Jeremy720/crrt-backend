CRRT Simulator API

The CRRT Simulator API provides test cases for use with the CRRT Simulator.

Example Images

GET cases

To get all cases, use the URL:

https://crrt-backend.herokuapp.com/api/v1/cases

Example Response:

Endpoint will return an array of all existing cases.

POST new case

To post a new case, use the URL:

https://crrt-backend.herokuapp.com/api/v1/cases

Example Response:

Endpoint will return new case's ID and a message indicating its successful creation.

PUT existing case

To put (edit) an existing case, use the URL:

https://crrt-backend.herokuapp.com/api/v1/cases/:id

Example Response:

Endpoint will return a message indicating successful edit of specified case.

DELETE existing case

To delete an existing case, use the URL:

https://crrt-backend.herokuapp.com/api/v1/cases/:id

Example Response:

Endpoint will return specified case's ID and a message indicating its successful deletion.
